# Flux-Agent 产品规范

## 产品愿景
基于 LangGraph + LLM 的通用 Agent 编排框架。让开发者通过 JSON 配置即可快速构建复杂 AI 工作流，大幅降低 LLM 应用开发门槛。

## 核心目标

### 1. 降低开发门槛
- 无需编写大量代码，JSON 配置即可定义工作流
- 内置 10+ 种通用节点开箱即用
- 统一的节点开发规范，降低扩展成本

### 2. 快速验证
- 配置即运行，支持快速迭代原型
- 支持子图嵌套和复用
- 丰富示例覆盖常见场景

### 3. 稳定 + 通用
- 成功指标：框架稳定可靠、API 向后兼容、节点覆盖主流场景
- 基于成熟的 LangGraph/LangChain 生态构建
- 模块化设计，按需安装依赖

## 目标用户
- Python 开发者
- 需要快速构建 LLM 工作流的团队
- AI 应用原型开发人员

## 使用场景
- 多步骤 AI 处理流程编排
- RAG 知识库问答系统
- Agent 智能体工作流
- 自动化数据处理管道
